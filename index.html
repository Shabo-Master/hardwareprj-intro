基于ONetSwitch的家庭网关

基于ONetSwitch开发板的家庭网关，能通过锐捷身份认证，具有远程状态监控功能。将其应用在使用锐捷认证的高校，能实现多设备上网，为使用者带来便利。

简介

该成品具有多个网口，连上互联网之后可作为具有NAT功能的网关使用。同时具有锐捷认证的能力，在使用锐捷认证计费方案的高校具有很高的便利性。另外还具有通过互联网远程状态监控的程序。

成品硬件上使用的是ONetSwitch20开发板。ONetSwitch开发板是基于Xilinx公司的Zynq-7000 ARM-FPGA芯片设计的：

  • 外设十分丰富，具有五个GE口，非常适合作为家庭网关使用
  • ARM-FPGA芯片，具有性能强大的ARM-a9核，作为家庭网关使用绰绰有余
  • ARM-FPGA芯片，硬件上可编程，有一定自主改变硬件的能力
  • 该平台母板是一块ZedBoard，有Xilinx官方资料加成，网上资料丰富

使用嵌入式Linux，OpenWrt的开源项目：

  • OpenWrt作为开源的Linux发行版，获取十分方便
  • 该项目广泛用于家庭路由器，有相当多的功能为家庭路由器设计，给我们的项目带来相当多的便利
  • 提供相当多的软件包供扩展
  • 其官方支持Zynq-7000芯片

成品功能分为三个部分，网关功能、锐捷认证功能、远程状态监控功能。

整个成品非常适合在学校使用，实现自动认证，多设备上网，远程管控。使用时，仅需将网关连上校园网接口，插上电源即可开机自动认证，将上网设备连上网关的GE口即可直接上网，使用管控客户端即可在电脑上监控设备的CPU负载、内存占用、网络等状态，并且对其进行控制。

网关功能实现

使用ONetSwitch官方固件和Bootloader作为启动文件，文件系统使用的是LEDE(OpenWRT)的开源项目提供的文件系统，以此搭建了一个嵌入式Linux系统，编译Linux内核开启Vlan和iptables功能，实现NAT功能。

锐捷认证

锐捷认证是锐捷公司私有的认证协议，其认证客户端并不对外开源，因此我们使用的是第三方的Linux认证客户端MentoHUST。使用LEDE提供的工具链单独编译

远程状态监控

远程状态监控能通过网络远程对网关的状态进行获取，并且能对其进行控制。使用Python实现，通过socket套接字传输信息。

技术

嵌入式Linux

嵌入式Linux（Embedded Linux）是一类嵌入式操作系统的概称，这类型的操作系统皆以Linux内核为基础，被设计来使用于嵌入式设备。

ONetSwtich板卡

ONetSwitch板卡是由南京MeshSr公司开发的，以Xilinx公司的Zynq系列芯片作为处理核心的专门用来开发交换机应用的开发板。开发板有着丰富的网络外设资源，可以进行灵活的扩展和二次开发。

OpenWrt

OpenWrt是适合于嵌入式设备的一个Linux发行版。相对原厂固件而言，OpenWrt不是一个单一、静态的固件，而是提供了一个可添加软件包的可写的文件系统。这使用户可以自由的选择应用程序和配置，而不必受设备提供商的限制，并且可以使用一些适合某方面应用的软件包来定制你的设备。

Python

Python，是一种面向对象、直译式的电脑程序语言。它包含了一组功能完备的标准库，能够轻松完成很多常见的任务。
